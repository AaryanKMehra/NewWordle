# NewWordle

**Concept:** A Wordle-style word guessing game that features an "Infinite Mode." Instead of a single daily word, players receive a new word immediately upon a successful guess.

**Core Mechanics:**
- **Infinite Play:** Successfully guessing a word generates a new target word and clears the board without a page refresh.
- **Score Tracking:** The score increments for each correct word found.
- **Reset Logic:** If the player fails to guess the word within 6 attempts, the total score resets to 0.

**Aesthetic:**
A high-end, minimalist monochromatic palette (Black, White, and Grey) to ensure a clean and professional user interface.

**Tech Stack:**
- HTML5 (Structure)
- CSS3 (Grid-based layout and minimalist styling)
- Vanilla JavaScript (State management and game logic)
